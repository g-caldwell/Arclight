# Project Arclight

Project Arclight is a cataloging and management software intended for backyard astronomers. This project allows a user to store, sort, 
and manipulate many different celestial bodies, all while allowing them to be sorted, searched through and saved. This 
makes cataloging and tracking celestial bodies much easier. Arclight has two main functions:
managing, and journaling. The manager is intended to allow a user to work with structured data in a graphical user interface that 
can be sorted and edited by the user. The journal function acts as an actual journal, allowing the user to save any 
notes or interesting observations that may have occured. In this way, you have the added benefit of keeping 
a database of celestial bodies while still maintaining the functionality of a notebook.
